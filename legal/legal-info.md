# Legal Information

**Service Operator**  
Unfiltered UK is operated by an independent developer collective.  
No corporate entity is involved.

**Jurisdiction**  
This platform operates under UK law. All disputes subject to English courts.

**Contact**  
For legal requests: legal@unfiltereduk.dev (PGP encrypted preferred)

**Transparency**  
- Source code: public on GitHub  
- Architecture: documented in `/help_center`  
- No backdoors. No secret access.  
- Server owners hold their own data — **we cannot reset, delete, or view it**

**DMCA / Content Removal**  
We only process takedowns for:
- Global DMs
- Platform-level user profiles
- Bot listings

**Server content must be addressed with the server owner directly.**

---
Unfiltered UK exists to empower communities — not to control them.
